структура папок
node_modules - хранятся все необходимые для функционирования реакта пакеты (их очень много и они зависят друг от друга)
public       - хранится статика это файлы которые не меняются(это логотипы штмл итд)
src          - тут хранится сам реакт


public/index.html - <div id="root"></div> - место где подключен реакт

src/index.js - является точкой входа в реакт приложение

удаление неиспользуемой информации

можно снести все в папке src оставив только App.js index.js

если снести пакеты node_modules для того что бы меньше места занимало то что
бы сбилдить заново проект нужно npm i или npm install
