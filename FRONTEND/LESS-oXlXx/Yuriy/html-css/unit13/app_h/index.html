План действий
  • Создать упорядоченый список а, b, с... внутри элемента fieldset, в котором будет описан план действий.
  • Добавить на страницу неупорядоченый список ul в котором будут перечислены преимущества форматирования по умолчанию для html элементов.
  • Поместить на странице горизонтальную линию hr.
  • Разместить после горизонтальной линии элемент списка определений dl, в котором дать определение терминам html, css u javascript.
  • Вставить элемент progress после нашего списка определений dl. Затем поставить разрыв строки br, что-бы визуально отделить контент на нашей странице.
  • Попросить пользователя выбрать язык программирования который он хочет изучить. К элементу ввода input подключить элемент списка данных datalist.
  • Написать листинг функции на javascrpt, которая выводит в консоль числа от 0 до 9. Использовать элемент code.
  • Записать математическое равенство элиптическои кривой. Использовать элемент var.
  • Выписать химическую формулу этилена.
  • Спросить у пользователя, является ли он роботом. Использовать элемент fildset.
  • Выполнить все пункты плана.

Преимущества:

  • Скорость создания HTML страницы
  • Отсутсвие необходимости создавать CSS селекторы и добавлять стили, захламляя тем самым таблицу стилей
  • Упрощение читаемости HTML разметки
